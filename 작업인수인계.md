# 한자 발음 블로그 색인 작업 인수인계 메모

## 작업 배경

`korean-pronunciation.blogspot.com` 블로그의 한자 발음 색인 페이지(`index.html`)를
SEO 최적화를 위해 4개 파일로 분할하는 작업을 완료했다.
GitHub Pages에는 Vite로 빌드된 검색 도구가 별도로 운영 중이다.

---

## 완료된 작업

### 1. Blogspot용 4분할 색인 (완료 ✅)

원본 `index.html` (7,028줄, ~1.2MB, 링크 약 11,000개)을 4개 파일로 분할.

| 파일명 | 블로그스팟 슬러그 | 초성 | URL |
|--------|-----------------|------|-----|
| `index_ga.html` | `index-g-d` | ㄱ·ㄴ·ㄷ | `https://korean-pronunciation.blogspot.com/2026/05/index-g-d.html` |
| `index_ra.html` | `index-r-b` | ㄹ·ㅁ·ㅂ | `https://korean-pronunciation.blogspot.com/2026/05/index-r-b.html` |
| `index_sa.html` | `index-s-ng` | ㅅ·ㅇ | `https://korean-pronunciation.blogspot.com/2026/05/index-s-ng.html` |
| `index_ja.html` | `index-j-h` | ㅈ·ㅊ·ㅋ·ㅌ·ㅍ·ㅎ | `https://korean-pronunciation.blogspot.com/2026/05/index-j-h.html` |

> 슬러그 표기 기준: 국립국어원 로마자 표기법  
> g(ㄱ), d(ㄷ), r(ㄹ), b(ㅂ), s(ㅅ), ng(ㅇ), j(ㅈ), h(ㅎ)

분할 스크립트: `D:\download\split_index.py`

### 2. 링크 동작 방식 (완료 ✅)

| 링크 종류 | 동작 |
|---------|------|
| Blogspot 색인 nav → 다른 색인 | 현재 창 |
| Blogspot 색인 → 개별 포스트 | 새 탭 (`target="_blank"`) |
| GitHub Pages nav → Blogspot 색인 | 새 탭 (`target="_blank"`) |

## 미완료 작업 (다른 컴퓨터에서 해야 할 일)

### Vite 프로젝트 소스에 nav 추가

**목적**: `npm run deploy` 시 nav가 자동으로 포함되도록 소스에 추가

**방법**: Vite 프로젝트의 `index.html` (루트 또는 `/public/index.html`)에  
`<body>` 태그 바로 다음에 아래 코드 삽입:

```html
<p style="font-family:'Noto Sans KR',Arial,sans-serif;font-size:0.95em;margin-bottom:16px;border-bottom:1px solid #ddd;padding-bottom:12px;">
  📚 <strong>초성별 상세 색인 (블로그)</strong> &nbsp;
  <a href="https://korean-pronunciation.blogspot.com/2026/05/index-g-d.html" target="_blank" rel="noopener noreferrer" style="color:#1a6fbf;font-weight:700;text-decoration:none;margin-right:10px;">[ㄱ~ㄷ]</a>
  <a href="https://korean-pronunciation.blogspot.com/2026/05/index-r-b.html" target="_blank" rel="noopener noreferrer" style="color:#1a6fbf;font-weight:700;text-decoration:none;margin-right:10px;">[ㄹ~ㅂ]</a>
  <a href="https://korean-pronunciation.blogspot.com/2026/05/index-s-ng.html" target="_blank" rel="noopener noreferrer" style="color:#1a6fbf;font-weight:700;text-decoration:none;margin-right:10px;">[ㅅ~ㅇ]</a>
  <a href="https://korean-pronunciation.blogspot.com/2026/05/index-j-h.html" target="_blank" rel="noopener noreferrer" style="color:#1a6fbf;font-weight:700;text-decoration:none;">[ㅈ~ㅎ]</a>
</p>
```

**삽입 위치 예시**:
```html
<body>
  <!-- ↑ 이 바로 아래에 위 코드 삽입 -->
  <div id="app"></div>
  ...
```

**배포 명령**:
```bash
npm run deploy
```
→ 자동으로 Vite 빌드 후 gh-pages 브랜치에 push됨
